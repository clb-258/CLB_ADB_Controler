# CLB_ADB_Controler
CLB ADB 控制工具是一款开源多功能的安卓设备控制工具。它基于 Android SDK 中的 ADB 命令行工具制作。
<br>
用户可以通过本工具，实现设备信息查看、设备实时控制以及应用/文件管理等功能。
<br><br>
<p>软件基于 Python 开发，并且完全开源。</p>
<p>您可以直接下载编译好的程序，在 Windows 下执行即可运行。</p>
<p>当然了，如果你要直接从源码运行，请先在您的 Python 中安装这些外置库：</p>
<p>  - PyQt5</p>
<p>  - pywin32</p>
